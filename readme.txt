Easily Enable WP Auto Updates for WordPress Core Plugins and Themes.

== Description ==

This plugin will automatically do WordPress core updates and also update themes and plugins, this includes
Automatic updates for maintenance and security updates. Updates will be installed automatically.


== Installation ==

1. Upload to the /wp-content/plugins/ directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.

== Frequently Asked Questions ==

= Where can I find the Admin Settings =

Admin Settings for this plugin is a top-level menu "Auto Updates" in the WordPress Admin Dashboard.

= Will this work with my current theme =

Yes, this plugin will work with any WordPress theme.
